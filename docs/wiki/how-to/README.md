# How-to Guides（ハウツーガイド）

**問題解決指向** - 特定の課題を解決するための実践的なガイドです。

ハウツーガイドは、「〜したい」という具体的な目的に対して、その実現方法を段階的に説明します。チュートリアルとは異なり、特定のトピックに焦点を当てています。

## ガイドの使い方

1. **目的を明確にする**: 何を実現したいか具体的に考える
2. **該当するガイドを選ぶ**: 目的に合ったガイドを選択
3. **手順に従う**: ガイドの手順を順番に実行
4. **カスタマイズする**: 自分のニーズに合わせて調整

## よくある質問

### Q: チュートリアルとハウツーガイドの違いは？

**A**: チュートリアルは全体的な学習を目的とし、ハウツーガイドは特定の課題を解決することを目的としています。

| | チュートリアル | ハウツーガイド |
|-|--------------|--------------|
| 目的 | 学習 | 問題解決 |
| 範囲 | 広い | 狭い |
| 順序 | 順番に実行 | 必要な部分だけ |

### Q: 複数のガイドを組み合わせることはできますか？

**A**: はい。たとえば、「プロンプトをカスタマイズする」と「画像品質を最適化する」を組み合わせることで、より高度なカスタマイズが可能です。

### Q: ガイドに従っても上手くいかない場合は？

**A**: 以下を試してください。

1. **トラブルシューティング**: 各ガイドの「トラブルシューティング」セクションを確認
2. **リファレンス**: [API リファレンス](../reference/api.md)で詳細を確認
3. **Issue**: GitHub の Issue で質問
